1. Реализовать класс Product который состоит из: id, название, цена
2. Реализовать класс Cart который состоит из списка продуктов добавленных к данной корзине
3. Реализовать класс ProductDao с поддержкой операции: удаление, добавление, поиск по id, получение доступных продуктов. Данный класс взаимодействует с БД
4. Реализовать класс СartDao с поддержкой операции: удаление, добавление, поиск по id. Данный класс взаимодействует с БД
5. Создать БД с поддержкой таблиц: Products, Carts
6. Создать JdbcTemplate для установления соединения с БД
7. Создать Mappers для приведения данные SQL-формы в Java-объект
