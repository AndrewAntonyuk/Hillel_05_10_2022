X 1.1 Создать класс PetrolStation. Класс имплементризрует заправочную станцию.



X 1.2 Класс PetrolStation содержит свойство вещественного типа  с именем amount. Данное свойство представляет количество топлива на станции.



X 1.3 В классе PetrolStation реализовать метод doRefuel. Метод принимает значение вещественного типа. При выполнении данного метода идет вычитание запрашиваемого топлива из доступного количества.



X 1.4 Одна заправка занимает время от 3-10 секунд. После истечении времени ожидания, осуществляет вычитание значения из п.5



X 1.5 Заправочная станция может одновременно обслуживать не более 3 запросов.
