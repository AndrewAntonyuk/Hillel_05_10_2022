1. Создать базу данных Student c атрибутами: id, name, email.

2. Создать сущность Student и добавить в нее аннотации. Поле id должно заполняться автоматически. Данная сущность должна отражать представление таблицы Student из БД.

3. Создать конфигурационный класс HibernateSession, в котором указать свойства для подключения к БД и список классов с аннотациями Entity.

4. В классе HibernateSession добавить статический метод для получения объект типа SessionFactory.

5. Создать DAO-класс для хранения операций: добавления, удаления, изменения сущности, получения всех записей и конкретной по ID.

6. Добавить N-записей в таблицу Student.

7. Проверить работоспособность приложения, выполнить методы по удалению, изменению, добавлению записей, а также выборки одной и всех записей.
